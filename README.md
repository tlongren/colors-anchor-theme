#Colors

A colorful, minimal (and responsive) theme for [AnchorCMS.](http://anchorcms.com)

---

##Usage

1. Copy the theme folder into your Anchor site's `themes` directory via FTP
2. Activate the theme in the Admin area -> Extend -> Metadata
3. Enjoy!
